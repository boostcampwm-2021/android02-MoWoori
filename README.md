## 한줄소개

- 각자 미션을 수행하여 스티커 판을 채워나가며 함께 눈사람을 성장시키는 습관형성 도움 앱

## 기대효과

- 깃헙의 잔디채우기와 비슷하게 목표들 달성한 것을 가시적으로 볼 수 있다.
- 목표 달성에 대한 동기부여를 줄 수 있다.

## 기능소개

- 사용자가 이루고자 하는 미션을 등록하여 원하는 목표치의 스탬프를 지정하면 이에 해당하는 스티커판을 무료로 제공!
- 여러 명의 구성원 서비스 제공!
- 단일 그룹이 아닌 멀티 그룹 기능 제공!
- 물질적 보상은 아니지만 귀여운 눈사람 키우기 기능 제공!



## 주요기능

### 📌 구글 로그인 (+ 자동 로그인, 추천 닉네임 & 프로필 사진 등록)

<img width="300" alt="스크린샷 2021-12-03 오전 2 31 14" src="https://user-images.githubusercontent.com/68374234/144473202-aa1f5f0d-456e-45e0-afef-bf18d3425734.png">

> **로그인 화면에서는 구글 계정을 통해 로그인을 할 수 있습니다.**

- 권한 설정
  - 처음 앱을 실행하면 사진 및 미디어 액세스 권한을 묻게 됩니다.
- 프로필 사진
  - 로그인에 성공하면 유저는 프로필 사진을 설정할 수 있습니다.
- 프로필 닉네임
  - 닉네임은 앱에서 추천 해주고 사용자가 직접 정할 수도 있습니다.
- 자동 로그인 기능도 있습니다.

  - 눈사람 팔, 표정 애니메이션

> **내가 수행할 미션을 추가할 수 있습니다.**

- 미션명 입력

  - 15자 이내로 정합니다.

- 미션기간 설정

  - 종료일이 시작일보다 빠르지 않도록 합니다.

  - 10, 20, 30번 중 하나로 정할 수 있습니다.

  - '완료'버튼 클릭 시 유효성 검사를 실행합니다.

> **인증사진, 한줄평과 함께 미션을 인증할 수 있으며, 해당 그룹원들에게 미션을 인증했다는 FCM 알람이 전송됩니다.**

- 인증사진 업로드

  - 카메라에서 직접 사진을 찍어 업로드 할 수 있습니다.
  - 갤러리에서 사진을 선택해서 업로드 할 수 있습니다.
  - 사진을 선택하지 않으면 기본 이미지가 업로드 됩니다.

- 한줄평 입력

  - 5~100자 사이로 작성합니다.

- FCM 알림 전송

  - 미션 인증 시, 해당 그룹원들에게 푸시알람이 전송됩니다.
  - 푸쉬알람을 클릭해서 앱을 열면 미션 인증 상세화면을 볼 수 있습니다.
